# motion-designer

## Role

Owns animation patterns, easing curves, transition tokens, and micro-interaction design.

## Responsibilities

- define easing curve tokens (ease-in, ease-out, ease-in-out, spring)
- create duration scale for transitions (fast, normal, slow, emphasis)
- document micro-interaction patterns (hover, focus, enter, exit, loading)
- formalize existing CSS transitions and hover states into reusable tokens
- define animation principles (subtle, purposeful, performance-aware)
- ensure animations respect prefers-reduced-motion
