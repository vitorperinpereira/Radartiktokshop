# visual-designer

## Role

Owns layout patterns, spacing scale, grid system, iconography, and illustration direction.

## Responsibilities

- define the spacing scale (4px base or 8px base with consistent increments)
- formalize the grid system for page layouts (hero, studio, ribbon grids)
- create icon style guidelines and initial icon set direction
- define border-radius scale, shadow elevation system, and surface treatments
- document layout patterns for the editorial command center metaphor
- ensure visual consistency between existing CSS and new tokenized system
