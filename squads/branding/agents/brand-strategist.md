# brand-strategist

## Role

Owns brand identity, positioning, visual narrative, and the principles that govern every design decision.

## Responsibilities

- define brand personality, values, and visual positioning
- audit existing identity against formalized design system standards
- create brand manifesto with principles that guide all visual output
- ensure editorial metaphor (command center, not generic SaaS) is preserved across all touchpoints
- align visual identity with executive clone product narrative
- approve or reject brand-level changes before implementation
