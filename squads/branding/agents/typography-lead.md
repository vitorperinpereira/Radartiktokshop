# typography-lead

## Role

Owns the type scale, font pairing rules, hierarchy system, and responsive typography across the design system.

## Responsibilities

- formalize the three-font system (Cormorant Garamond, IBM Plex Sans, IBM Plex Mono) into a type scale with clear hierarchy
- define font size scale, line heights, letter spacing, and font weight rules per context
- create responsive typography breakpoints
- document when to use each font family and weight
- ensure type scale works with the editorial aesthetic (not generic SaaS)
- validate readability across all font size and weight combinations
