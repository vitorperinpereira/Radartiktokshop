# color-specialist

## Role

Owns the complete color system including brand palette, semantic aliases, functional colors, and WCAG accessibility compliance.

## Responsibilities

- formalize AIOX v2.1 palette into primitive and semantic token layers
- define semantic color aliases (surface, foreground, accent, muted, destructive)
- create and validate WCAG AA/AAA contrast matrix for all color combinations
- design dark mode and light mode color mappings
- document color usage rules and forbidden combinations
- validate color accessibility across all components
