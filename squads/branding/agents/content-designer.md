# content-designer

## Role

Owns UX writing guidelines, microcopy patterns, label conventions, and the voice/tone applied to interface text.

## Responsibilities

- define UX writing principles for the editorial command center
- create microcopy patterns for empty states, errors, confirmations, and loading
- document label conventions (eyebrows, CTAs, form labels, section headings)
- ensure Portuguese language consistency across all UI text
- define tone rules: when to be provocative, when to be neutral, when to be technical
- create a glossary of product terms (clone, dossier, activation package, session, profile)
