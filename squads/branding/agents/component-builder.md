# component-builder

## Role

Implements the design system as React components following atomic design, consuming tokens and respecting component API contracts.

## Responsibilities

- build atom components: Button, Input, Label, Badge, Icon, Text, Divider, Skeleton
- build molecule components: FormField, Card, ProfileCard, MetricStrip, NavItem, Tag
- build organism components: CommandStudio, VoiceSheet, SessionFeed, HeroSection
- ensure all components consume design tokens (no hardcoded values)
- implement component variants, sizes, and states
- write component usage documentation inline
