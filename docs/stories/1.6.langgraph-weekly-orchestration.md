## Story

**As a** project operator,
**I want** the weekly bootstrap to execute through a LangGraph state graph with retries,
**so that** the scoring pipeline has a real orchestration layer before AI agents are introduced.

## Acceptance Criteria

1. A typed weekly-run state model exists under `services/orchestration/state`.
2. A LangGraph weekly graph exists under `services/orchestration/graphs` and orchestrates input validation, deterministic feature extraction, and completion state updates.
3. The feature extraction node uses LangGraph retry policy for transient failures.
4. The weekly-run service delegates execution to the LangGraph graph while preserving `pipeline_runs` persistence behavior.
5. Tests cover graph execution, retry behavior, and the CLI weekly-run flow.
