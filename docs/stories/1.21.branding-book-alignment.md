## Story

**As a** product operator and future designer,  
**I want** the TikTok Scrapper branding book to reflect the real product naming, visual system, and frontend surfaces already present in the repository,  
**so that** future design and implementation work can use one reliable brand source of truth instead of mixing stale artifacts from another product.

## Acceptance Criteria

1. `squads/branding/output/branding-book.md` defines the naming architecture for the existing repository, product, workspace, and UI labels using current repo artifacts only.
2. The branding book maps the current palette, key surfaces, and signature component patterns to the real Vite frontend and current Streamlit/API surfaces without claiming non-existent assets or implementation paths.
3. The branding book records the current implementation gaps and a pragmatic rollout order for future branding work.
